# 隐私政策（PRIVACY）

本扩展 Edge AI Translator 仅在本地浏览器环境运行，不收集、不出售任何个人信息。所有翻译数据仅在你的浏览器与您配置的 Provider（例如 OpenAI 兼容接口或自建 API）之间传输。

版本：v0.1.0
生效日期：2025-11-29

一、数据范围
- 文本内容：当你主动触发“整页翻译/划词翻译/测试翻译”时，页面中的可翻译文本或你输入的文本将被发送到你在选项页配置的 Provider。
- 元信息：可能包含源/目标语言、模型名称、工作流参数（风格/语气/术语表/占位符保护/响应格式/最小长度）、批处理拆分信息（仅在提示中声明，不包含页面结构或 URL）。
- 防风控噪声（可选）：如启用“随机噪声注入”，会向提示中附加一段“请忽略”的随机噪声文本，用于降低请求相似度，不含可识别的个人信息。

二、数据处理与传输
- 处理位置：扩展在本地浏览器中收集文本并经由 HTTPS 调用你配置的 Provider 接口。
- 直连 Provider：若配置为 openai-compatible，将调用 /v1/chat/completions；若配置为 custom，将向你的自定义 JSON 接口 POST {inputs[], sourceLang, targetLang, ...}。
- 第三方政策：翻译数据受相应 Provider 的隐私政策与服务条款约束（如 OpenAI、云厂商或你的自建服务）。请确保合规。

三、存储与缓存
- 配置：保存在 chrome.storage.sync（随账户同步），包括 endpoint、apiKey（如填写）、headers、模型、工作流、行为与高级参数。
- 运行期缓存：后台 service worker 内置 LRU+TTL 缓存，仅存储“输入文本→译文”对，驻留内存，随浏览器/扩展生命周期清空；不做持久化。
- 本扩展不将翻译内容或日志持久化到磁盘，也不向开发者或第三方服务器回传。

四、权限说明
- host_permissions: <all_urls>：为实现全站整页/划词翻译与 UI 注入所需。
- permissions: storage, contextMenus, scripting, activeTab：用于配置保存、右键菜单、脚本/样式注入与当前标签页交互。
- 可选优化：你可以在选项中降低限速与批量，减少对 Provider 的调用频率。

五、用户控制
- 选项页：可完全关闭缓存、关闭批处理、调整最小长度与启用/禁用噪声注入。
- 取消与中止：整页翻译可一键取消，立即中止排队与在途请求。
- 跳过策略：代码块/表单/隐藏元素等默认不采集；极短或仅符号的段落会跳过。

六、数据共享
- 除你配置的 Provider 外，本扩展不向任何第三方披露、出售或共享数据。
- 开发者不运营任何服务器端收集服务。

七、安全实践
- 仅通过 HTTPS 调用 Provider；内置指数退避与限速，降低被风控导致的错误放大。
- 建议你使用代理网关加固：额外限速、鉴权与审计；避免在公共设备保存 API Key。

八、儿童隐私
- 本扩展不面向 13 岁以下儿童；不主动收集任何敏感个人信息。

九、政策更新
- 如有重大变更，将在仓库 README 与版本发布说明中标注。

十、联系
